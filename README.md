# 🎨 Advanced Responsive Color Generator

## 🔗 Important Links

- **Live Website:** [color-code-gen.vercel.app](https://color-code-gen.netlify.app/)
A sleek, lightweight, and interactive color generator built with pure HTML, CSS, and Vanilla JavaScript. This tool allows developers and designers to generate random colors, pick custom colors, and instantly copy Hex or RGB codes to their clipboard.

## ✨ Features

* **Full-Screen Preview:** The entire page background updates dynamically to preview your selected color.
* **🎲 Random Generator:** Generate a brand new, random hex color with a single click.
* **📋 Click-to-Copy:** Click on the Hex or RGB text to instantly copy the code to your clipboard.
* **📱 Fully Responsive:** Adapts beautifully to mobile and desktop screens using modern CSS Grid (`auto-fit`).
* **🌈 Custom Color Picker:** Integrates the native HTML5 `<input type="color">` for precise color selection.
* **🏷️ Preset Swatches:** Quick access to a curated grid of popular, vibrant UI colors.

## 🚀 Built With

* **HTML5:** Semantic structure and custom `data-*` attributes.
* **CSS3:** Flexbox, CSS Grid, smooth transitions, and responsive design.
* **Vanilla JavaScript:** DOM manipulation, Event Delegation, Clipboard API, and HEX/RGB math logic. No external libraries or frameworks required!

## 🛠️ How to Run

Because this project uses vanilla web technologies, there is no build process or package manager required.

1. Clone this repository or download the source code.
2. Navigate to the project directory.
3. Double-click the `index.html` file to open it in your default web browser.
4. *Alternatively, you can use a tool like VS Code Live Server for hot-reloading.*

## 📖 Usage Instructions

1. **Pick a Color:** Click the circular color wheel to open your system's native color picker.
2. **Generate Random:** Click the large "Generate Random Color 🎲" button to let the math do the work.
3. **Use Presets:** Click any of the color swatches at the bottom of the card to jump to that color.
4. **Copy Code:** Hover over the `#HEX` or `rgb()` text. Click it once, and it will confirm the code has been copied to your clipboard.
